Notion Printer 사용 가이드

Notion에서 export한 HTML 또는 ZIP을 인쇄하기 좋은 통합 문서로 만들고, 출력 전 브라우저 미리보기에서 페이지 나눔과 이미지 크기를 직접 다듬는 도구입니다.

실행 앱Ubuntu 앱 메뉴의 Notion Printer
추천 설정Compact Print + Fast Preview 추가

추천 처음 쓰는 경우 옵션을 거의 바꾸지 않아도 됩니다. 기본값으로 생성한 뒤 미리보기에서 필요한 부분만 다듬으세요.

1. 기본 사용 흐름

  1. Notion Printer를 실행합니다.
  2. Notion에서 export한 HTML 또는 ZIP 파일을 1개 또는 여러 개 선택합니다.
  3. 추천 설정을 확인하고 생성을 시작합니다.
  4. 브라우저 미리보기에서 페이지 상태를 확인합니다.
  5. 여기서 시작, 앞에 붙이기, 이미지 크기 조절로 다듬습니다.
  6. 브라우저 인쇄에서 PDF 저장 또는 실제 인쇄를 진행합니다.
추천 이미지나 첨부가 많은 문서는 ZIP 파일을 그대로 선택하는 편이 더 안정적입니다.

주의 목차는 긴 통합 문서일 때만 켜는 것을 권장합니다. 문서가 짧거나 1개뿐이면 목차 없는 출력이 더 깔끔합니다.

중요 삭제는 출력본 편집에만 적용됩니다. 원본 Notion 문서는 삭제되지 않습니다.

4. 설치 방법

더블클릭 설치

  1. notion-printer_1.0.1_all.deb 파일을 다운로드합니다.
  2. 파일을 더블클릭합니다.
  3. Ubuntu Software 화면에서 Install을 누릅니다.
  4. 업데이트 후에는 컴퓨터를 다시 시작해주세요.
  5. 다시 시작한 뒤 Ubuntu 앱 메뉴에서 Notion Printer를 실행합니다.

터미널 설치

sudo apt install ./notion-printer_1.0.1_all.deb
안내 업데이트 후에는 컴퓨터를 다시 시작해주세요.

5. 파일 선택 규칙

선택 방식결과
HTML 1개 선택통합 파이프라인으로 단일 문서 출력
HTML 여러 개 선택선택한 문서들을 하나로 통합
ZIP 1개 선택압축을 풀고 원본 HTML을 찾아 출력
ZIP 여러 개 선택선택한 ZIP들을 순서대로 통합
HTML + ZIP 함께 선택함께 통합 가능
주의 이미지나 첨부가 많은 문서는 ZIP 파일을 그대로 선택하면 원본 구조를 함께 유지하기 쉽습니다. 생성된 *_print*.html 파일은 입력용이 아닙니다.

6. 여기서 시작

여기서 시작은 선택한 블록부터 새 페이지를 시작하게 만드는 기능입니다.

상황해결
제목만 이전 페이지 맨 아래에 남음제목에서 여기서 시작
표가 페이지 중간부터 애매하게 시작표에서 여기서 시작
이미지와 설명이 서로 다른 페이지로 갈라짐설명 또는 이미지 앞에서 여기서 시작
  1. 새 페이지로 보내고 싶은 제목, 문단, 표, 이미지 근처에 마우스를 올립니다.
  2. 나타나는 여기서 시작 버튼을 누릅니다.
  3. 미리보기가 다시 계산되면 원하는 위치에서 페이지가 시작되는지 확인합니다.
주의 여기서 시작을 많이 쓰면 페이지 수가 늘어날 수 있습니다. 페이지 수를 줄여야 하면 앞에 붙이기나 앞 페이지로 붙이기를 같이 사용하세요.

7. 합치기 기능

앞에 붙이기

앞에 붙이기는 현재 블록을 바로 앞 내용과 이어 붙입니다.

사용 상황설명
짧은 문단이 다음 페이지에 혼자 있음앞 내용과 이어 붙여 페이지 낭비 줄이기
여기서 시작을 취소하고 싶음다시 앞 내용에 붙이기
앞 페이지에 빈 공간이 있음현재 블록을 위로 당기기

앞 페이지로 붙이기

앞 페이지로 붙이기는 현재 페이지 전체를 앞 페이지 쪽으로 당겨 붙입니다.

사용 상황설명
현재 페이지가 거의 비어 있음앞 페이지와 합쳐 페이지 수 줄이기
앞 페이지에 빈 공간이 큼현재 페이지 시작 부분을 앞으로 이동
빈 페이지가 생김앞 페이지로 붙여 제거
주의 이미지나 표가 너무 크면 붙지 않을 수 있습니다. 이때는 이미지 크기를 줄인 뒤 다시 시도해 주세요.

8. 이미지 조절

이미지 위에 마우스를 올리면 이미지 조절 도구가 나타납니다.

버튼역할
이미지 크기슬라이더로 이미지 비율 조정
선택 추가너비를 맞출 이미지들을 선택
현재 기준 통일현재 이미지 폭을 기준으로 선택 이미지 너비 맞춤
폭 적용입력한 px 값으로 이미지 폭 적용
기본값이미지 크기를 100%로 복원
팁 큰 이미지 때문에 합치기가 안 되면, 먼저 이미지를 조금 줄인 뒤 앞 페이지로 붙이기를 다시 눌러보세요.

10. 인쇄/PDF 저장

  • 브라우저의 머리글 및 바닥글 옵션은 끄는 것을 권장합니다.
  • 배율은 기본값부터 확인합니다.
  • PDF 저장은 브라우저 인쇄 창에서 Save to PDF를 선택합니다.

11. 결과 파일 위치

결과는 첫 번째로 선택한 HTML 또는 ZIP 파일이 있는 폴더 아래에 생성됩니다.

_notion_printer_integrated_factory/

실행할 때마다 시간별 폴더가 만들어집니다.

_notion_printer_integrated_factory/run_20260423_170000/
파일설명
*_print_compact_fast.html빠른 미리보기용 Compact 출력본
*_print_compact.htmlCompact 출력본
factory_summary.json생성 요약 정보
merged_source/통합된 원본 HTML 정보
안내 ZIP 파일을 선택한 경우에는 원본 구조를 정리해 두는 _notion_printer_imported_sources/ 폴더가 함께 보일 수 있습니다.

12. 문제 해결

문제확인할 것
설치가 안 됨sudo apt install ./notion-printer_1.0.1_all.deb로 설치
앱 메뉴에 안 보임잠시 기다리거나 로그아웃 후 다시 로그인. 터미널에서는 notion-printer 실행
HTML 또는 ZIP 선택 후 실패원본 Notion export HTML 또는 ZIP인지, 생성된 *_print*.html 파일이 아닌지, ZIP은 압축을 풀지 않아도 되는지, 접근 권한이 있는지 확인
이미지가 안 보임가능하면 ZIP 파일을 선택하고, HTML을 직접 선택했다면 주변 이미지/첨부 폴더가 함께 있는지 확인
문제 공유가 필요함사용한 HTML 또는 ZIP 파일명과 생성 시간 폴더명을 함께 전달
