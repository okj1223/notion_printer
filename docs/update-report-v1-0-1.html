이번 버전에서 달라진 점

1. HTML뿐 아니라 ZIP도 바로 넣을 수 있습니다

파일 선택창에서 .html과 .zip을 모두 받을 수 있게 바뀌었습니다. ZIP은 앱이 자동으로 풀고, 안에서 원본 Notion HTML을 찾아서 이어서 처리합니다.

2. 여러 ZIP도 한 번에 통합할 수 있습니다

ZIP 1개만 처리하는 수준이 아니라, ZIP 여러 개를 선택해 순서대로 하나의 출력본으로 통합할 수 있습니다. HTML 여러 개, HTML+ZIP 혼합 선택도 함께 지원합니다.

3. 이미지가 빠지거나 경로를 못 찾는 상황을 더 줄였습니다

이미지 파일 경로가 조금 어긋난 경우에도 원본 구조를 기준으로 다시 찾도록 보강했습니다. 특히 ZIP을 그대로 선택하면 원본 폴더 구조가 함께 유지돼 이미지 누락 가능성을 줄이는 데 도움이 됩니다.

4. 잘못된 입력을 다시 넣는 실수를 막았습니다

이미 생성된 *_print*.html 파일을 입력으로 다시 선택하면 안내 메시지를 보여주고, 원본 Notion export HTML 또는 ZIP을 선택하도록 유도합니다.

5. 앱 아이콘과 배포 패키지를 팀 사용 기준으로 정리했습니다

앱 메뉴 아이콘을 Notion Printer 전용 아이콘으로 맞췄고, 팀에 전달하는 설치 파일도 notion-printer_1.0.1_all.deb 기준으로 다시 빌드했습니다.

6. 팀원용 가이드를 HTML 기준으로 정리했습니다

사용 가이드는 이제 공유하기 쉬운 HTML 기준으로 정리되어 있습니다. 팀원 입장에서는 설치 파일, 선택 방법, 결과 위치, 문제 해결 내용을 한 파일에서 바로 볼 수 있습니다.

팀원이 바로 체감하는 사용 흐름

이전처럼 하기 쉬웠던 행동 이제는 이렇게 하면 더 편합니다
압축을 풀고 HTML만 따로 찾아 선택 ZIP 파일 그대로 선택
문서 여러 개를 합칠 때 HTML만 골라서 정리 ZIP 여러 개나 HTML+ZIP을 그대로 선택 순서대로 넣기
이미지 깨짐이 걱정되어 폴더 구조를 따로 신경 씀 가능하면 ZIP을 그대로 선택해서 원본 구조를 함께 유지
예전에 만든 출력 HTML을 다시 넣어보는 실수 이제 앱이 걸러 주므로 원본 HTML 또는 ZIP만 선택하면 됨
안내 결과 파일은 첫 번째로 선택한 HTML 또는 ZIP 파일 폴더 아래 _notion_printer_integrated_factory/에 생성됩니다.

권장 사용 팁

이미지가 많은 문서

HTML보다 ZIP 파일을 그대로 선택하는 편이 더 안정적입니다.

문서 여러 개를 한 번에 출력

선택한 순서대로 통합되므로, 합치고 싶은 순서로 파일을 고르면 됩니다.

목차 사용

짧은 문서는 목차를 끄고, 긴 통합 문서만 목차를 켜는 것을 권장합니다.

입력 파일 선택

생성된 *_print*.html 파일은 다시 입력으로 사용하지 말고 원본 export 파일을 선택해 주세요.

업데이트 방법

  1. notion-printer_1.0.1_all.deb 파일을 받습니다.
  2. 아래 두 가지 방법 중 하나로 설치합니다.
방법 설명
삭제 후 더블클릭 기존 앱을 삭제한 뒤 notion-printer_1.0.1_all.deb를 더블클릭해서 다시 설치합니다.
삭제 없이 터미널 업데이트 기존 앱을 지우지 않고 터미널에서 바로 업데이트합니다. 
sudo apt install ./notion-printer_1.0.1_all.deb

설치 후 컴퓨터를 다시 시작한 뒤, 아래 명령으로 적용된 버전을 확인합니다.

dpkg-query -W -f='${Version}\n' notion-printer
업데이트 설치 안내 더블클릭 후 Ubuntu Software가 이미 설치됨 또는 회색 Install 버튼만 보여주면 기존 앱을 삭제하고 더블클릭으로 다시 설치하거나, 삭제하지 않고 위 터미널 명령으로 업데이트하세요. 설치가 끝나면 컴퓨터를 다시 시작해주세요.

간단 확인 체크리스트

  • 앱 메뉴에서 Notion Printer가 새 아이콘으로 보이는지 확인
  • HTML 1개 또는 ZIP 1개 선택 시 정상 생성되는지 확인
  • ZIP 여러 개 선택 시 한 문서로 통합되는지 확인
  • 이미지 많은 문서를 ZIP으로 넣었을 때 미리보기가 자연스러운지 확인
  • 결과 파일이 _notion_printer_integrated_factory/ 아래 생기는지 확인
문제 공유 시 사용한 HTML 또는 ZIP 파일명과 생성 시간 폴더명을 함께 알려주면 확인이 훨씬 빨라집니다.
