Notion Printer 1.0.4 업데이트 노트

1.0.4는 Fast 품질의 기본값을 빠름 기준으로 맞춘 버전입니다. 앱에서 처음 실행했을 때, 터미널로 실행했을 때, 팀원용 안내 문서를 볼 때 모두 같은 기본값을 기준으로 확인할 수 있습니다.

한눈에 보기

기본값 정리 Fast 품질 기본값이 빠름 (1200px / WEBP 68)로 표시됩니다.
앱과 CLI 일치 앱 실행 화면과 터미널 실행 기본값이 같은 기준을 사용합니다.
문서 기준 통일 홈, 다운로드, 처음 쓰는 법, 릴리즈 가이드의 기본 품질 설명을 맞췄습니다.
이전 수정 유지 1.0.3의 ZIP 내부 부분 HTML 제외 처리는 그대로 유지됩니다.

변경사항

1. Fast 품질 기본값을 빠름으로 맞췄습니다

앱을 처음 열면 Fast 품질이 빠름 (1200px / WEBP 68) 기준으로 시작합니다. 이미지가 많은 문서도 미리보기를 가볍게 열어 확인하기 위한 기본값입니다.

2. 터미널 실행 기본값도 같은 기준으로 정리했습니다

별도 옵션 없이 스크립트를 실행할 때도 --max-edge 1200, --quality 68 기준을 사용합니다. 앱으로 실행하든 터미널로 실행하든 기본 출력 조건이 달라지지 않습니다.

3. 선명 옵션은 고급 선택지로 유지합니다

더 선명한 이미지가 필요하면 고급 옵션에서 균형, 선명, 고품질을 선택할 수 있습니다. 기본 시작점만 빠름으로 정리했습니다.

4. 팀원용 안내 문서를 같은 말로 맞췄습니다

홈 화면, 다운로드 페이지, 처음 쓰는 법, 릴리즈 가이드에서 기본 품질을 같은 값으로 안내합니다. 버전과 설치 파일명도 1.0.4 기준으로 맞췄습니다.

달라지는 사용 흐름

상황 1.0.4 기준 안내
기본값 그대로 출력 Fast 품질 빠름 (1200px / WEBP 68)로 미리보기를 생성합니다.
이미지를 더 선명하게 확인 고급 옵션에서 선명 또는 고품질을 선택합니다.
터미널로 실행 별도 품질 옵션이 없으면 --max-edge 1200, --quality 68 기준으로 실행됩니다.
ZIP 입력 사용 1.0.3에서 정리한 ZIP 내부 부분 HTML 제외 처리를 그대로 사용합니다.

설치와 확인

  1. 업데이트/다운로드 페이지에서 notion-printer_1.0.4_all.deb를 받습니다.
  2. 삭제 후 재설치 또는 터미널 업데이트 중 하나를 선택합니다.
  3. 설치가 끝나면 컴퓨터를 다시 시작합니다.
  4. 다시 시작한 뒤 설치된 버전을 확인합니다.
cd ~/Downloads

# 삭제 후 다시 설치할 때
sudo apt remove notion-printer

# 삭제하지 않고 업데이트할 때
sudo apt install ./notion-printer_1.0.4_all.deb
dpkg-query -W -f='${Version}\n' notion-printer
권장 확인 설치된 버전이 1.0.4로 표시되는지 확인하고, 앱의 고급 옵션에서 Fast 품질 기본값이 빠름 (1200px / WEBP 68)로 보이는지 확인하세요.

간단 확인 체크리스트

  • dpkg-query 결과가 1.0.4로 표시되는지 확인
  • 앱 실행 시 Fast 품질 기본값이 빠름 (1200px / WEBP 68)인지 확인
  • HTML 또는 ZIP 파일 1개로 출력본이 생성되는지 확인
  • 여러 HTML/ZIP 파일 선택 시 통합 출력본이 생성되는지 확인
