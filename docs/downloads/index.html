기존 1.0.0이 설치된 PC에서 1.0.1로 올리는 방법은 두 가지입니다.

  1. 기존 앱을 삭제한 뒤 notion-printer_1.0.1_all.deb를 더블클릭해서 다시 설치합니다.

cd ~/Downloads
sudo apt install ./notion-printer_1.0.1_all.deb
