Notion Printer
사용 가이드 업데이트 노트 다운로드
팀 배포용 Notion 출력 도구

Notion export를 인쇄하기 좋은 통합 문서로 바꿉니다.

Notion에서 내려받은 HTML 또는 ZIP을 선택하면 Notion Printer가 이미지 경로를 정리하고, 여러 문서를 한 권처럼 합친 뒤 브라우저 미리보기와 인쇄용 HTML을 생성합니다.

무엇을 해결하나요?

ZIP 그대로 선택

압축을 풀고 HTML과 이미지 폴더를 따로 찾는 과정을 줄입니다.

여러 문서 자동 통합

HTML 여러 개, ZIP 여러 개, HTML+ZIP 혼합 입력을 한 출력본으로 묶습니다.

출력 전 미리보기 편집

페이지 시작 위치, 앞 페이지 붙이기, 이미지 크기 조절을 브라우저에서 바로 합니다.

기본 사용 흐름

앱 실행

Ubuntu 앱 메뉴에서 Notion Printer를 엽니다.

파일 선택

Notion export HTML 또는 ZIP을 1개 이상 선택합니다.

옵션 확인

기본값은 Compact + Fast, 목차 끄기, 쪽번호 켜기입니다.

미리보기/인쇄

브라우저 미리보기에서 다듬고 PDF 저장 또는 실제 인쇄를 진행합니다.
