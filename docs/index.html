팀에서 같이 쓰는 Notion 출력 도우미

Notion export, 인쇄 준비까지 한 번에 끝내요.

HTML이나 ZIP을 고르면 Notion Printer가 이미지 경로를 정리하고, 여러 문서를 한 권처럼 묶은 뒤 바로 확인할 수 있는 인쇄 미리보기를 만들어줍니다.

최신 설치 파일 받기 처음 쓰는 법 보기

이전 버전은 보관함에 따로 모아두었습니다.

업데이트 설치 안내

설치 파일명은 배포마다 달라집니다. 아래 예시의 1.x.x 자리에 다운로드한 파일명에 적힌 버전을 넣어 실행하세요.

  1. 기존 앱을 삭제한 뒤 받은 .deb 파일을 더블클릭해서 다시 설치합니다.
  2. 기존 앱을 삭제하지 않고 터미널에서 바로 업데이트합니다.
# 삭제 후 다시 설치할 때
sudo apt remove notion-printer

# 삭제하지 않고 업데이트할 때
sudo apt install ~/Downloads/notion-printer_1.x.x_all.deb

설치가 끝나면 컴퓨터를 다시 시작해주세요.

다시 시작한 뒤 설치된 버전은 아래 명령으로 확인합니다. 다운로드한 파일명에 적힌 버전과 같으면 정상입니다.

dpkg-query -W -f='${Version}\n' notion-printer

여러 문서를 한 권으로

HTML 여러 개, ZIP 여러 개, HTML+ZIP 혼합 입력도 선택한 순서대로 묶습니다.

마지막 손질은 미리보기에서

페이지 시작 위치, 앞 페이지 붙이기, 이미지 크기 조절을 브라우저에서 바로 합니다.
